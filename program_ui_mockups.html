Program Manager · UI Concepts
Five non-functional mockups exploring how to guide a user through the key steps — build a spacecraft → assemble a fleet vehicle → plan a mission → launch & fly — while keeping Rocket Playground's existing dark / mono aesthetic. Each concept takes a different guidance philosophy.
01

Countdown

Linear guided wizard

Concept: a launch-countdown metaphor. The whole program is one sequence — T‑4 → T‑0 — and the rail across the top is both the navigation and the progress tracker. Completed steps glow and check off; the current step is locked-in with a focused work panel and a helper sidebar. You always know exactly where you are and what's next. Best for first-time users who want a hand-held path.

02

Mission Hub

Status dashboard

Concept: a non-linear home screen. Four big status cards (one per stage of the workflow) show what you've built at a glance, and a "recommended next step" banner + a glowing card nudge you forward without forcing an order. Great for returning users who want to jump straight to the part they care about while still being guided.

03

Flight Checklist

Pre-flight checklist

Concept: the program reads like a real pre-flight checklist down the left rail — grouped, with ◉ / ○ / ✓ status dots. Click an item to expand its editor on the right in place. It rewards completion (everything turns green) and makes prerequisites obvious. Feels authentically aerospace and is self-documenting.

04

Trajectory

Pipeline flow

Concept: the workflow drawn as a trajectory of connected nodes — Spacecraft → Fleet → Mission → Launch → Fly — with arrows that light up as data flows downstream. It visually teaches the dependency chain (a mission needs a fleet vehicle, which needs a spacecraft). Click a node to drop into its workspace below. Echoes the mission Node Map you already have.

05

Co-Pilot

Persistent assistant

Concept: keep the full command-center workspace, but dock a persistent Co‑Pilot panel on the right that always shows the single most useful next action, contextual tips for whatever you're touching, and one-tap quick actions. Guidance lives beside the work instead of gating it — invisible to experts, a lifeline to newcomers. Could later become a real assistant.
