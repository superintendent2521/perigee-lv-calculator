# Credits & Attributions

## LV Calculator (`lv_calc.html`)

**Silverbird M** — orbital mechanics reference implementation  
The delta-V budget math, staging equations, and orbital insertion models are
derived from / cross-checked against Silverbird M by Adam Getchell.  
https://silverbirdastronautics.com/

**Natural Earth** — world map data  
The launch-site map uses 1:110m land polygon data from Natural Earth,
released under a public domain (CC0) license.  
https://www.naturalearthdata.com/
