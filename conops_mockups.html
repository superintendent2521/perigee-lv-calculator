Nonfunctional interaction mockup
Body-centric Tracks Each body gets a horizontal lane. Vehicles travel through lanes over time. Transfers are vertical connections. Matches CONOPS diagrams Easy to add events per body Time axis shared — crowded for long missions
Continuous Altitude (Log Scale) Y axis = physical distance from Earth, log-scaled. Transfers are diagonal. Physically honest. Physically accurate 8 orders of magnitude — hard to read Crowded near Earth
Node Map (Subway / Mission Graph) Bodies and orbits are nodes. Edges show ΔV cost. User traces a mission path through the graph. Not time-based. Great for planning ΔV budgets Immediately shows what's possible No time axis — boiloff, LAN windows invisible Multi-vehicle missions hard to show
